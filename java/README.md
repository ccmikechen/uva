# Java

## Execute program

```shell
$ ./run PROBLEM_NUMBER

# Example

$ ./run 100
1 10 20
100 200 125
201 210 89
900 1000 174
```

## Run test

```shell
$ ./test PROBLEM_NUMBER

# Example

$ ./test 100
calc(1, 10) passed.
calc(100, 200) passed.
calc(201, 210) passed.
calc(900, 1000) passed.
calc(1, 10000) passed.
```

## Check result

```shell
$ ./check PROBLEM_NUMBER

# Example

$ ./check 100
Passed!
Elapsed time: .067046821
```
